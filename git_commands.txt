1> git init: For initialize new repository

2> git add <file name>:to add file for used in next commit

3> git commit -m "commit message":to commit the file with message 

4> git status:show the current status of the files that which are not commited yet

5>git push origin master :send changes file to master remotly

6> git checkout <branch name>:switch from one branch to another branch

7> git checkout -b<branch name>: if branch not exist then create branch and switch

8>git pull:fetch and merge changes on server to present working directory

9> git log :shows the commit id along with all the history and time stamp

10> git remote add origin<server>: to connect remote server to be able topush to it

11>git reverse --hard:to revert the change in file
